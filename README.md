# Student Record System

## Description

The Student Record System is a beginner-friendly Python project designed to manage student information efficiently. It demonstrates the use of Python dictionaries, functions, loops, and conditional statements to perform basic CRUD (Create, Read, Update, Delete) operations.

Users can add new student records, update existing details, search for a student by name or ID, delete records, and display all stored student information. This project is ideal for practicing core Python concepts and understanding how simple data management systems work.

## Features

- Add new student records
- Update student information
- Search for a student
- Delete student records
- Display all student records
- Menu-driven interface
- Input validation for better user experience

## Technologies Used

- Python 3
## Learning Outcomes

- Working with dictionaries
- Creating menu-driven applications
- Implementing CRUD operations
- Writing modular code using functions
- Handling user input
